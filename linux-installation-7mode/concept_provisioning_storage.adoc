= 스토리지 프로비저닝
:icons: font
:imagesdir: ../media/


[role="lead"]
UNIX용 SnapDrive를 사용하면 호스트의 명령줄 옵션을 사용하여 스토리지를 프로비저닝하고 기존 스토리지 시스템에 접속할 수 있습니다. Data ONTAP를 사용하여 스토리지를 프로비저닝할 수도 있습니다.
