= umount 명령의 문제로 인해 RHEL/OEL 5.x에서 스냅 관리 작업이 실패합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
"umount" 명령의 문제로 인해 RHEL/OEL 5.x에서 스냅 관리 작업이 실패합니다. 이를 해결하려면 명령줄 인터페이스에서 '-o vers=3'을 사용하여 NFS 엔터티를 마운트해야 합니다.
