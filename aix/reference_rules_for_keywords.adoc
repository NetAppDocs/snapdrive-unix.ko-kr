= 키워드에 대한 규칙입니다
:icons: font
:imagesdir: ../media/


[role="lead"]
UNIX용 SnapDrive에서는 키워드를 사용하여 UNIX용 SnapDrive 작업의 대상을 지정합니다.

SnapDrive for UNIX는 키워드를 사용하여 작업 중인 호스트 및 스토리지 시스템 객체에 해당하는 문자열 시퀀스를 지정합니다. 다음 규칙은 UNIX용 SnapDrive 키워드에 적용됩니다.

* 각 키워드 앞에 하이픈(-)을 붙입니다.
* 키워드를 연결하지 마십시오.
* 약어가 아닌 전체 키워드 및 하이픈을 입력합니다.
