= 데몬을 강제로 중지합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
모든 명령이 실행될 때까지 기다리지 않으려면 데몬을 강제로 중지할 수 있습니다. 데몬을 강제로 중지하라는 요청이 수신되면 SnapDrive for UNIX 데몬이 실행 중이거나 대기열에 있는 모든 명령을 취소합니다. 데몬을 강제로 중지하면 시스템의 상태가 정의되지 않을 수 있습니다. 이 방법은 권장되지 않습니다.

루트 사용자로 로그인해야 합니다.
