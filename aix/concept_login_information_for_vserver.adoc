= SVM에 대한 로그인 정보
:icons: font
:imagesdir: ../media/


[role="lead"]
사용자 이름과 암호를 사용하여 UNIX용 SnapDrive에서 SVM에 액세스할 수 있습니다. 또한 vsadmin으로 로그인하는 것 외에도 UNIX용 SnapDrive를 실행하고 메시지가 표시될 때마다 올바른 사용자 이름과 암호를 제공해야 하므로 보안이 제공됩니다.
