= 호스트에서 LUN을 분리하기 위한 명령 구문입니다
:icons: font
:imagesdir: ../media/


[role="lead"]
"SnapDrive storage disconnect" 명령을 사용하여 지정한 LUN의 매핑을 제거하려면 다음 구문을 사용합니다.

'SnapDrive storage disconnect - lun long_lun_name [lun_name...]
