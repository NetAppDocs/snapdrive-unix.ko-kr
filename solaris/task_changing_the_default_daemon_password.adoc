= 기본 데몬 암호를 변경하는 중입니다
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX에는 나중에 변경할 수 있는 기본 데몬 암호가 할당되어 있습니다. 이 암호는 암호화된 파일에 저장되며 읽기 및 쓰기 권한이 루트 사용자만 할당됩니다. 암호를 변경한 후에는 모든 클라이언트 응용 프로그램에 수동으로 알려야 합니다.

루트 사용자로 로그인해야 합니다.

.단계
. 기본 암호 변경:
+
``스냅드라이브 암호 *’

. 암호를 입력합니다.
. 암호를 확인합니다.
