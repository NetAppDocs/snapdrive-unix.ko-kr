= 스토리지 시스템 목록 보기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
호스트 시스템에서 'SnapDrive config list' 명령을 사용하여 모든 스토리지 시스템을 볼 수 있습니다.

. 다음 명령을 입력합니다.
+
'* SnapDrive 구성 목록 * '

+
[listing]
----
snapdrive config list
username appliance name appliance type
-----------------------------------------------------
root      roaster     StorageSystem
----
