= 호스트 클러스터에서 노드 제거
:icons: font
:imagesdir: ../media/


[role="lead"]
호스트 클러스터에서 새 노드를 제거하려면 Solaris용 _Veritas Cluster Server 설치 가이드 의 지침을 따르십시오. _ LLT 및 Gab을 구성한 후 "SnapDrive storage disconnect" 명령을 사용하여 호스트 클러스터의 모든 공유 스토리지 엔터티에 대한 모든 LUN 매핑을 해제해야 합니다.
