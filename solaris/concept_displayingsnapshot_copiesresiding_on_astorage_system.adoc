= 스토리지 시스템에 상주하는 스냅샷 복사본을 표시합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive 명령을 사용하여 스토리지 시스템의 스냅샷 복사본을 표시할 수 있습니다.

스토리지 시스템에 상주하는 스냅샷 복사본에 대한 정보를 표시하려면 다음 구문을 사용합니다.

"SnapDrive snap show-filer filername[filername...] [-verbose]'
